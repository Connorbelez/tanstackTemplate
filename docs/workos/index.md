# AuthKit
Full-fledged authentication platform, from your first user to enterprise single sign-on.
[Go to Quick Start](https://workos.com/docs/authkit)
### [Hosted UI Customizable hosted UI for authentication at any size.](https://workos.com/docs/authkit/hosted-ui)### [Enterprise auth Single sign-on, passkeys, social sign-in, passwords, and more.](https://workos.com/docs/authkit/sso)### [Roles and Permissions Advanced RBAC with custom roles and role assignment.](https://workos.com/docs/authkit/roles-and-permissions)### [Radar Protect your app from bots, fraud, and abuse.](https://workos.com/docs/authkit/radar)### [Custom Metadata Store additional information about users and organizations.](https://workos.com/docs/authkit/metadata)### [Connect Built-in support for MCP and OAuth applications.](https://workos.com/docs/authkit/connect)
