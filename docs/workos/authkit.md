# Get started with AuthKit
Set up AuthKit in your app using either the WorkOS CLI or a manual framework-specific guide.
## On this page
  * [Install using the WorkOS CLI](https://workos.com/docs/authkit#install-using-the-workos-cli)
  * [Or install using your preferred stack](https://workos.com/docs/authkit#or-install-using-your-preferred-stack)

## Install using the WorkOS CLI
[](https://workos.com/docs/authkit#install-using-the-workos-cli)
Automatically integrate WorkOS into your app with this command. Learn more on [GitHub](https://github.com/workos/workos-cli).
WorkOS CLI
```

| 
npx workos@latest

---|---  

```

[](https://workos.com/docs/authkit#or-install-using-your-preferred-stack)
