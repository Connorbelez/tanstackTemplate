# Cash & Obligations Ledger Planning Package

This directory contains a Notion-ready planning package for the proposed
cash-and-obligations ledger goal.

Files:

- `GOAL-draft.md` - the page content and suggested properties for a new Goal in
  the FairLend Product Planning System.
- `SPEC-ledger-cash-and-obligations.md` - a fuller product and technical spec
  that can back the Goal page or be linked from it.
- `research-notes.md` - repo-grounded findings from GitNexus, local code review,
  and related Notion pages.

Notes:

- This package is intended to be source material for a Notion Goal page in the
  FairLend Product Planning System.
- If another `SPEC-ledger-cash-and-obligations.md` exists elsewhere in the
  repository, reconcile it with the version in this directory so there is a
  single canonical spec before implementation begins.
